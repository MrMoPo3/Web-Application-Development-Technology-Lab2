## Лабораторна робота

Назва лабораторної роботи: **Організація спільної роботи користувачів Web-додатка**

## Завдання

На основі реалізованого Web-додатку розробити функції спільної роботи користувачів за допомогою Django Channels:

- можливість передавати робочі дані додатку між користувачами у реальному часі;
- можливість для адміністратора переглядати список користувачів, які працюють з додатком у даний час;
- забезпечити тестування API та WebSocket-функцій.

## Склад репозиторію

- `backend/` - Python/Django/DRF/Django Channels код серверної частини;
- `backend/requirements.txt` - список пакетів, сформований командою `pip freeze > requirements.txt`;
- `docs/redoc.html` - HTML-документація API у форматі Redoc;
- `docs/postman-tests.js` - JavaScript-тести для Postman.

## Запуск

```powershell
cd backend
py -m pip install -r requirements.txt
py manage.py migrate
py manage.py runserver
```

Redoc-документація доступна у браузері через файл `docs/redoc.html`.

API запускається за адресою:

```text
http://127.0.0.1:8000/
```
